Study Buddy Tutorial

Overview

Study Buddy is a cross platform mobile application that provides GVSU students a way to find study groups and make  group chats for current classes they are taking.

Tech Stack

Flutter – Flutter is a framework created by Google that gives us the ability to write code in 1 language and generate native versions of iOS, Android, Windows, Linux, macOS and web. We chose it because it could save us time in development since we only have to write one version and it could generate native versions for all available platforms.

Firebase – Firebase is a cloud storage database solution created by Google. It’s easy to use and fits the scope of the project. The main point of Firebase is to store data on the cloud and provide real time communication between the client and the cloud without the need of a WebSocket server.

Conclusion

This is my implementation of Study Buddy in Flutter. Flutter is not the only native framework on the market right now. If you are more familiar with JavaScript and web frameworks, This may not be ideal for you, and you should try React Native. React Native is a web framework that can do the same as flutter but depends on JavaScript and styling of CSS html which some people consider to be easier than flutter approach. Moreover, to implement chat functionality, you could also create a WebSocket server and make it listen to user action rather than depend on Firebase Realtime database like my implementation. With WebSocket, you could implement notifications for when the user is currently typing and other good stuff. But WebSocket is way beyond the scope of this project therefore we decided not to use it now. Here is the source code of our group implementation of study buddy cross platform group study app.
